# AI Help Desk Sample App

## 준비 

Create an environment using venv

MAC / Linux
```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

Windows
```bash
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

## 로컬 실행 

```bash
flask run 
```
